List of Pseudo-classes

Pseudo-class Description
:hover Matches when the user hovers over an element.
:focus Matches when an element is focused (e.g., an input field).
:nth-child(n) Selects the nth child of a parent element.
:first-child Matches the first child of a parent element.
:last-child Matches the last child of a parent element.
:not(selector) Excludes elements matching the specified selector.
:checked Matches input elements that are checked (e.g., radio or checkbox).
:disabled Matches input elements that are disabled.
:enabled Matches input elements that are enabled.
:empty Matches elements that have no children.
:nth-of-type(n) Selects the nth element of its type among siblings.
:first-of-type Matches the first element of its type among siblings.
:last-of-type Matches the last element of its type among siblings.
:lang(language) Matches elements with a specific language attribute value.
:root Matches the root element of a document (e.g., ).
:link Matches unvisited links.
:visited Matches links the user has visited.
:active Matches the active element (e.g., a clicked link).
